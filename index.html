Node Utilities

This addon provides a pie menu that groups together several useful node operations.

It has a wide range of functions such as align, distribute, flip, node search, coloring, etc.

Versions 1.6.1 and above are compatible with Blender 3.4 and above.


How to use

  • Go to Compositor, Shader Editor or Geometry Node Editor.
    • Press D on them to open utility pie menu, or E to open selection pie menu.
    • Press Shift z or Shift x on them to change width of selected nodes.
    • Press Shift Ctrl Alt D on them to duplicate and unparent selected nodes.
  • Press Shift E on Geometry Node Editor to toggle viewer on and off.

Functions

Node Utility Pie Menu

The functions that can be used vary depending on the node or frame selected.

Align / Flip / Distribute

  • Align X: Align selected nodes along the x-axis(same as press S > X > 0).
  • Align Y: Align selected nodes along the y-axis(same as press S > Y > 0).
  • Flip X: Flip selected nodes' x location(same as press S > X > -1).
  • Flip Y: Flip selected nodes' x location(same as press S > Y > -1).
  • Distribute X: Arrange the selected nodes evenly along the x-axis.
  • Distribute Y: Arrange the selected nodes evenly along the y-axis.
Coloring
  • Set Fav Color: Changes the color of the selected nodes and frames to the color specified in preferences.
  • Use Color Off: Turn off Use Color for the selected nodes and frames.
  • Color Input: Changes the color of the selected nodes and frames to the its color.
  • Paste Color of Active Node: Applies the active node or frame color to all selections.
Labeling
  • Labeling with Node Name: Changes the label of a node depending on its name or operation.
    Math, Vector Math, Boolean Math, Compare, and Mix Color nodes are named according to their operation.
  • Label Size: Changes the label size of the selected frames.
    Min: Set to 8 / Half: Set to 36 / Max: Set to 64 / Number: Set to its number.
Connect Last to Output (Geometry Nodes only)
  • Connect Last to Output: Connects the specified node to the output.
    For large node groups, you can sometimes use Shift Alt LMB to connect intermediate states to the output to check them.
    This makes it easy to connect a last node back into an output node again.
  • Set Active Node to Last: Specify the active node as the Last Node.
Search Node

Search Node: Search a node by its name and the name of its connected output socket.
Search a node by the name of its output socket is useful when a node group has multiple Group Inputs.

Zoom

Zoom: Zoom to setected nodes or frames by zoom level specified in preferences.
This is useful because the period key does not zoom when you have a single node selected.
Note that it will not zoom exactly to the zoom level specified in the preferences.

Hide unconnected socket for same type

Hide unconnected socket for same type: Hide unconnected socket for same type of the active node.
This is especially useful if you have multiple group inputs.


Selection Pie Menu

Selects nodes in each direction from the mouse position when the key is pressed.
If you turn on Select Frame, it will also select the frame.


Others

  • Increase / Decrease Node Width: Shift x / Shift z keys will increase or decrease the width of the selected node.
    The shortcut keys and node width increment/decrement value can be changed in the preferences.
  • Toggle Viewer: Shift e toggles the viewer on and off (Geometry Nodes only)
    This works when there is only one viewer.
  • Duplicate and Unparent: Shift Ctrl Alt d to duplicate and unparent selected nodes.

Preferences

You can change the shortcut keys.

You can change the following items:

  • Width Additinal Amount: node width increment/decrement value.
    Node width increment/decrement values.
  • Select Frame: Whether to select frames in the Selection Pie Menu.
  • Fav Color: Colors used in Set Fav Color.
  • Auto Zoom: Whether to automatically zoom after searching a node.
  • Zoom Level: Higher values zoom more.

Installation

For Blender4.2 or higher
  • download zip file, do not unzip it.
  • go to Blender's user preference,open the "Add-ons" or "Get Extensions" tab
  • click "Install from Disk" from downward arrow at right up and select the zip file.
For Blender4.1 or lower
  • download zip file, do not unzip it.
  • go to Blender's user preference,open the Add-ons tab
  • click Install and select the zip file.
  • check the box next to its name to enable it.
